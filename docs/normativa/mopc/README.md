## Cómo sacar un extracto operativo

1. Si el PDF tiene texto extraíble 📄, usa `pdftotext -layout` para trabajar sobre el texto.
2. Si es escaneado 🖼️, aplica OCR primero:
   ```bash
   ocrmypdf <input>.pdf <output>-ocr.pdf -l spa
   ```
   (Requiere `ocrmypdf` con `tesseract-ocr-spa`.)
3. Sigue la plantilla de [`../mived/ccrd-vol-i.md`](../mived/ccrd-vol-i.md).
4. Commit: `docs(normativa): extracto operativo R-XXX`.

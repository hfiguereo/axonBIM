# AxonBIM icon style guide

## Purpose

This pack defines the first shared visual language for AxonBIM UI tools.
Icons are intentionally simple, vector-based and readable at 16-32 px.

## Visual rules

- Canvas: `32 x 32` SVG.
- Stroke: `2` px, rounded caps and joins.
- Base color: slate `#d8e2ef`.
- Primary accent: BIM blue `#38bdf8`.
- Action accent: amber `#f59e0b`.
- Avoid gradients, shadows and raster effects in tool icons.

## Naming

- Tools: `tool_<name>.svg`
- Actions: `action_<name>.svg`
- Status: `status_<name>.svg`
